Profile

Website ini dibuat sebagai media pembelajaran dan pengembangan kemampuan di bidang Web Development, khususnya dalam memahami dasar-dasar HTML, CSS, dan JavaScript. Proyek ini menjadi langkah awal dalam membangun pemahaman tentang bagaimana sebuah website dirancang, dikembangkan, dan ditampilkan secara profesional.


Dalam proses pembuatannya, website ini berfokus pada struktur dasar halaman (semantic HTML), pengaturan tata letak (layouting), desain visual menggunakan CSS, serta interaksi sederhana menggunakan JavaScript. Setiap bagian halaman dirancang untuk melatih pemahaman tentang elemen, atribut, dan form handling.


Selain itu, konsep responsive design juga diterapkan agar tampilan website dapat menyesuaikan berbagai ukuran layar, mulai dari desktop, tablet, hingga perangkat mobile. Hal ini penting karena saat ini mayoritas pengguna mengakses website melalui perangkat mobile.


Website ini juga menjadi sarana eksplorasi dalam memahami praktik terbaik pengembangan web modern, seperti penggunaan struktur kode yang rapi, pemisahan file HTML, CSS, dan JavaScript, serta optimasi tampilan agar tetap ringan dan cepat diakses.
